# js-express-practice
expressでメモアプリを作成しながら練習

## yarnのインストール
以下コマンドを実行(グローバルにインストール)
```
npm install -g yarn
```



##  Node.jsのインストール
https://nodejs.org/ja/ にアクセスしインストール

## Expressのボイラープレートを作成
下記コマンドを実行し、`express-generator` をインストール
```
npm install express-generator -g
```

下記コマンドを実行し、プロジェクトテンプレートを作成 
```
express memo-app --view=ejs
```
